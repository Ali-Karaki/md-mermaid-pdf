# md-mermaid-pdf Export (VS Code)

Export the active Markdown file to PDF using [md-mermaid-pdf](https://github.com/Ali-Karaki/md-mermaid-pdf). Mermaid diagrams are rendered correctly.

## Usage

1. Open a `.md` file
2. Run command: **Export Markdown to PDF (Mermaid)** (Cmd/Ctrl+Shift+P)
3. PDF is written alongside the markdown file

## Requirements

- [md-mermaid-pdf](https://www.npmjs.com/package/md-mermaid-pdf) installed globally (`npm i -g md-mermaid-pdf`) or available via `npx`
- Node.js ≥ 20.16
